Google Gemini API 키 등록

이 도구는 Google의 Gemini AI를 사용합니다. Gemini API는 무료이고 신용카드도 필요 없습니다. 키 발급은 30초면 끝나고, 한번 설정하면 다음부터는 바로 사용할 수 있습니다.

01
Google AI Studio (aistudio.google.com/apikey)로 이동해서 Google 계정으로 로그인합니다.
02
"Create API key" 버튼을 눌러 키를 발급받습니다. AIzaSy... 또는 AQ.... 로 시작하는 긴 문자열이 생성됩니다.
03
그 키를 복사해서 아래 입력칸에 붙여넣고 "등록"을 누르세요. 끝.
⚠ 절대로 비용이 나가지 않게 하는 법: Google Cloud 콘솔에서 결제(Billing) 정보를 등록하지 마세요. 결제 정보가 없으면 무료 한도(분당 15회·일 1,500회)를 초과해도 그냥 막힐 뿐, 절대 청구되지 않습니다.
키는 당신의 브라우저에만 저장됩니다. 다른 곳으로 전송되지 않습니다.
